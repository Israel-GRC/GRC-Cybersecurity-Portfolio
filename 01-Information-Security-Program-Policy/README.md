# Information Security Program Policy

## Project Overview

This project demonstrates the development of an Information Security Program Policy for Acme Financial Services, a fictional financial services organization.

The policy establishes governance requirements for protecting company systems, customer financial data, employee information, cloud services, applications, and business operations.

## Objectives

- Establish cybersecurity governance
- Define security responsibilities
- Support risk management activities
- Align with industry frameworks
- Protect confidentiality, integrity, and availability

## Framework Alignment

- NIST Cybersecurity Framework (CSF) 2.0
- ISO/IEC 27001

## Skills Demonstrated

- Information Security Governance
- Policy Development
- Risk Management
- Compliance Documentation
- Executive Communication
- GRC Program Management

## Artifacts

- Information Security Program Policy
- Document Control Register
- Revision History

## Disclaimer

This project was developed for educational and portfolio purposes. Acme Financial Services is a fictional organization.
